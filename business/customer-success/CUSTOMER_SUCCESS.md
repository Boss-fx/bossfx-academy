## Common Support Issues

### Download Problems
| Issue | Solution |
|---|---|
| "I didn't get my download email" | Check admin dashboard for order → resend via /api/admin?action=resend |
| "Download link expired" | Generate new token via /api/admin?action=token |
| "File won't download" | Verify token in URL, check Supabase Storage, generate fresh link |
| "Wrong product received" | Check order in Supabase, verify product_id, resend correct product |

### Payment Issues
| Issue | Solution |
|---|---|
| "I paid but got no confirmation" | Check Flutterwave dashboard → check Vercel webhook logs → manual fulfillment |
| "Payment failed" | Check Flutterwave transaction status, advise retry or different payment method |
| "I want a refund" | Evaluate via Flutterwave dashboard, process if within policy |
| "Double charged" | Check Flutterwave for duplicate transactions, initiate reversal |

### EA Issues
| Issue | Solution |
|---|---|
| "EA won't install" | Send MT5 installation guide, verify MT5 (not MT4) |
| "EA not trading" | Check recommended settings, verify broker compatibility, check market hours |
| "EA giving errors" | Request screenshot, check MT5 journal tab for error codes |

### Account Issues
| Issue | Solution |
|---|---|
| "Can't access Telegram" | Resend Telegram invite link |
| "Can't access VIP portal" | Verify VIP purchase, check /vip/ page access |
| "Want to upgrade" | Send payment link for next tier, apply value ladder |

## FAQ (For Customer-Facing Use)

**Q: Is BossFx Academy legitimate?**
A: Yes. We've been teaching forex since [founding]. Our EA is verified on the official MQL5 Marketplace. Join our free Telegram community to see real student discussions.

**Q: Do you guarantee profits?**
A: No one can guarantee trading profits. We guarantee quality education, proven strategies, and ongoing support. Your results depend on your discipline and risk management.

**Q: Can I get a refund?**
A: Digital products are non-refundable once downloaded. For mentorship, you can cancel before the next billing period. Contact us within 24 hours of purchase if you have concerns.

**Q: What payment methods do you accept?**
A: We accept debit/credit cards, bank transfers, USSD, and mobile money via Flutterwave. All payments are in Nigerian Naira (₦).

**Q: How do I install the EA?**
A: The EA works on MetaTrader 5 (MT5) only. Download the file from your email, open MT5, go to File → Open Data Folder → MQL5 → Experts, paste the file, restart MT5. Detailed guide included with purchase.

**Q: Can I access the course on mobile?**
A: Yes. All course materials are PDF/digital format accessible on any device. The website is mobile-optimized.
