## Growth Model

BossFx grows through a **content-led, community-amplified** loop:

```
Content → Discovery → Capture → Nurture → Purchase → Community → Referral
   ↑                                                                  │
   └──────────────────────────────────────────────────────────────────┘
```

The community drives referrals which drive more discovery. The more students succeed, the more content and social proof gets created, which attracts new students.

## Growth Loops

### Loop 1: Content → SEO → Leads (✅ Active)
```
Create blog post → Google indexes → Searcher finds → Reads → Downloads resource → Captured as lead → Drip email → Purchase
```
**Speed:** Slow (weeks to rank) but compounds over time
**Current status:** 11 blog posts, growing organic traffic

### Loop 2: Social → Engagement → Trust → Purchase (✅ Active)
```
Post trading content → Followers engage → Build trust → Visit website → Purchase
```
**Speed:** Medium (daily content, weekly conversions)
**Current status:** Active on Instagram, TikTok, YouTube, X

### Loop 3: Purchase → Community → Referral (✅ Partially Active)
```
Student buys → Joins Telegram → Engages → Tells friends → Friends buy
```
**Speed:** Medium (referral happens organically)
**Current status:** Telegram community active, but no formal referral program

### Loop 4: EA Marketplace → Brand → Full Product Suite (✅ Active)
```
Trader finds EA on MQL5 → Buys EA → Discovers BossFx Academy → Buys course/mentorship
```
**Speed:** Slow (marketplace-dependent)
**Current status:** EA live on MQL5 ($49.99), website has full product suite

### Loop 5: Free Tools → Leads → Purchase (✅ Active)
```
Trader searches for tool → Finds BossFx resource → Uses tool → Captured as lead → Drip → Purchase
```
**Speed:** Medium (SEO-dependent)
**Current status:** 8 free tools with lead capture forms
