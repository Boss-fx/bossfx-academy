# Telegram Integration — BossFx Academy

> **Last Updated:** 2026-06-30
> **Owner:** Engineering / Marketing

---

## Purpose

Telegram serves as the primary community platform for BossFx Academy students. All paid products include Telegram community access.

---

## Configuration

| Setting | Value |
|---|---|
| Community invite link | `https://t.me/qD_fBeaziqE5YzU8` |
| Location in code | `config.js → BFX.config.socials.telegram` |
| Also referenced in | `lib/products.js` (telegramInvite field for each product) |
| Also referenced in | `lib/email.js` (footer link in all emails) |

No environment variables required — Telegram integration is link-based only.

---

## How It Works

### Post-Purchase Community Access
1. Customer purchases any product
2. Fulfillment email includes "Join the Community" button linking to Telegram
3. Product-specific emails include `product.telegramInvite` link
4. All products currently share the same Telegram invite link

### Website References
- Footer social links on all pages
- Community page (community.html)
- Mentorship booking confirmation emails

### Booking Communication Preference
`api/booking.js` captures the student's preferred communication method. Options include:
- Telegram (default)
- WhatsApp
- Zoom

The ICS calendar invite generated by `lib/calendar.js` uses the selected method.

---

## Dashboard Settings

No Telegram Bot API is currently integrated. The integration is purely invite-link based.

To manage the Telegram group:
1. Open Telegram → Group Settings
2. Manage invite links, member permissions, admin roles
3. Pin important messages for new members

## Testing

Verify the invite link works by opening `https://t.me/qD_fBeaziqE5YzU8` in a browser.

---

## Common Failures

| Failure | Cause | Fix |
|---|---|---|
| Invite link expired | Telegram group settings changed | Generate new invite link in Telegram group settings, update `config.js` and `lib/products.js` |
| Link not in email | Email template missing Telegram CTA | Check `lib/email.js` for product-specific templates |
